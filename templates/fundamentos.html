Fundamentos de Python

1. Estruturas de Seleção

Permitem executar blocos de código com base em condições.

if x > 0:
    print("Positivo")
elif x < 0:
    print("Negativo")
else:
    print("Zero")

2. Estruturas de Repetição

Permitem repetir blocos de código.

for i in range(5):
    print(i)

3. Vetores e Matrizes

vetor = [1, 2, 3]
matriz = [[1, 2], [3, 4]]

4. Funções e Procedimentos

def saudacao(nome):
    print(f"Olá, {nome}!")

5. Tratamento de Exceções

try:
    num = int(input("Digite um número: "))
except ValueError:
    print("Entrada inválida!")
